Guess The Hidden Character Game Console App


  • Name:
    Guess The Hidden Character Game
  • Description:
    - This is a Console App developed using C# Programming Language.
    - When the user first fires up the app, it will show the title/logo of the app and prompt the user to enter his name. After it has the user's name, it will welcome the user. Then, it will show the app menu which is 3: Start, Instructions and Quit, while prompting the user to choose from those as number from 1 to 3. When the user chooses number 1, it will start the game. First, it will show to the user 3 characters with each name and attributes. Then, the screen will change and will show only 1 character with full-packed attributes and name existed as blank, while prompting the user to guess the name of the character based on the attributes. When the user guesses wrong, his 3 game life will reduce, otherwise his score/points will increase by one. And the user can play the game so long as he wants.
    Back to menu, when the user chooses number 2, it will show to the user the instructions for playing the app. And if the user chooses number 3, the app will thank the user and exit gracefully.

Student Record Management Console App


  • Name:
    Student Record Management (CLI-based System)
  • Description:
    - This is a Console App or CLI-based System (Command-line Interface -based System) developed using C# Programming Language.
    - The app starts directly with its logo name and menu stacked together, while the system prompting the user to choose from menu. If the user wants to add a student record to the system, he needs to choose number 1. Else if the user wants to view all the records recorded in the system, he needs to choose number 2. Else if the user only wants to know if a certain record is recorded in the system, he needs to choose number 3. And the last number is for exit/stop of the system.
    In the Add Section, the user first needs to choose the specific Student Course he wants to record or return back to menu. Then the user can enter the student record information. After clicking enter, the record is being recorded. The user can record student record again.
    In the View Section, the user can view all the records recorded in the specific student course which was should be specified by the user upon entering the section. Then, the user can return back to menu.
    In the Search Section, the user can enter the name of a student he wants to ensure if it is recorded in the system. If the student's name existed, the system will echo "Record Found!!!!", otherwise "Record Not Found!!!!"
  • Year:
